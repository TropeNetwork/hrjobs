2.3. PEAR

You need also some PEAR packages:

  • Config >= 1.10.3

  • DB >= 1.6.8

  • HTML_QuickForm >= 3.2.4

  • HTML_Template_Sigma >= 1.1.2

  • LiveUser >= 0.13.2

You can get the packages by installing PEAR. After that install the requiered packages by using the following command:

# pear install <package_name>

The package LiveUser is still a beta version, so the use is on your own risk. To install it use this command

# pear install LiveUser-beta
