3.4. Configure your webserver

Point the DocumentRoot to the public_html directory of your installation. Make sure you enabled php.

Example 3.1. sample Apache configuration

<VirtualHost *:80>
    ServerName hrjobs.example.com
    <Location /server-info>
      SetHandler server-info
      Order deny,allow
      Deny from all
      Allow from .cross.intern
    </Location>

    DocumentRoot /var/www/hrjobs/public_html
 
    php_admin_value include_path .:/var/www/hrjobs/public_html:/usr/lib/php

    AliasMatch ^/docs/(.*) /home/gerrit/workspace/hrjobs/public_html/doc/$1
    ErrorLog /var/www/hrjobs/log/error_log
    CustomLog /var/www/hrjobs/log/access.log common
</VirtualHost>
