3.3. Configuration

Change in your installation directory.

# cd hrjobs

Coopy the config-dist.xml file in the config directory to config.xml

# cp config-dist.xml config.xml

Open the config.xml in your preferred editor and edit the user, password, host and database entries to your database.

<?xml version="1.0" encoding="ISO-8859-1"?>
<conf>
  <database>
    <type>your_db_type</type>
    <user>your_user</user>
    <pass>your_password</pass>
    <host>your_host</host>
    <name>your_database</name>
  </database>
</conf>
